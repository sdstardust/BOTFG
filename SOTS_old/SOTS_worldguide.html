วิธี 1 — iframe (ง่ายที่สุด)

เซฟไฟล์นี้ไว้ในโฟลเดอร์เดียวกันกับหน้าหลัก แล้ววางโค้ดนี้ในตำแหน่งที่ต้องการ

<iframe src="SOTS_worldguide.html" width="100%" height="1400px" frameborder="0" style="border:none;" ></iframe>

วิธี 2 — แผนที่อย่างเดียว (SVG snippet)

คัดลอก SVG ด้านล่างไปวางตรงในหน้า HTML ที่ต้องการ แล้วเพิ่ม CSS font ด้านบน

<!-- ใส่ใน <head> ของหน้าปลายทาง --> <link href="https://fonts.googleapis.com/css2? family=Cinzel:wght@400;500& family=Sarabun:wght@300;400& display=swap" rel="stylesheet">

จากนั้นคัดลอกบล็อก <svg class="map-svg" ...>...</svg> จากไฟล์นี้ไปวางในหน้าปลายทาง

วิธี 3 — ผังตัวละครอย่างเดียว (HTML+JS snippet)

ต้องการสามส่วนจากไฟล์นี้ คัดลอกทั้งหมดไปไว้ในหน้าปลายทาง

1. CSS ทั้งหมดในบล็อก /* ── FILTER BUTTONS ── */ ถึง /* ── HINT ── */ 2. HTML ทั้งหมดในแท็ก <section id="characters">...</section> 3. JavaScript ฟังก์ชัน charFilter() และ buildCharGraph() ทั้งหมด
